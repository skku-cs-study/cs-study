## 🚨 Rules

- 매 `화` `15:00` 까지 - 각자 맡은 키워드/주제별로 PR 작성
  - 개념 정리
  - 예상 질문 (최소 3개, 답변은 작성자가 개인적으로 정리 후 스터디 시간에 공유)
- 스터디 진행 전까지 다른 사람들 PR 확인 후 코멘트 작성
  - 예상 질문 풀이 (필수)
  - 의문점 or 토의 필요한 부분
  - 추가하고 싶은 내용
- 매 `목` `13:00 ~ 15:00` - 스터디 진행
  - PR 작성자 답변 공유
  - 추가적인 PR 코멘트에 대해 토의
